# 套牌模板功能说明

## 概述
新增的套牌模板功能允许用户查看和管理炉石传说的套牌模板数据。

## 功能特点

### 1. 功能选择界面
- 应用程序首页现在提供两个功能选项：
  - **版本对比**：原有的卡牌版本对比功能
  - **套牌模板**：新增的套牌模板管理功能

### 2. 套牌模板功能

#### 数据读取
系统会读取以下三个JSON文件：
- `DECK_TEMPLATE.json` - 套牌模板定义
- `DECK.json` - 套牌信息
- `DECK_CARD.json` - 套牌中的卡牌

#### 数据关联
- 通过 `DECK_TEMPLATE` 的 `m_deckId` 关联到 `DECK` 的 `m_ID`
- 通过 `DECK_CARD` 的 `m_deckId` 关联到 `DECK` 的 `m_ID`
- 通过 `DECK_CARD` 的 `m_cardId` 获取卡牌ID

#### 功能列表
1. **版本选择**：选择要查看的游戏版本
2. **数据加载**：自动加载并关联套牌数据
3. **套牌列表**：以卡片形式展示所有套牌
4. **搜索过滤**：
   - 按套牌名称搜索
   - 按职业过滤
   - 按游戏模式过滤（标准/狂野）
5. **套牌详情**：点击套牌查看完整信息，包括：
   - 基本信息（ID、职业、类型等）
   - 完整卡牌列表
6. **数据导出**：将套牌数据导出为JSON格式

## 使用流程

1. 启动应用程序
2. 在首页选择"套牌模板"功能
3. 选择要查看的版本
4. 点击"加载套牌"按钮
5. 查看套牌列表，可使用搜索和过滤功能
6. 点击任意套牌查看详细信息
7. 可选择导出套牌数据

## 技术实现

### 文件结构
```
src/renderer/
  ├── index.html              # 首页（已更新，添加功能选择）
  ├── deck-template.html      # 套牌模板页面
  ├── deck-template.js        # 套牌模板逻辑
  └── deck-template.css       # 套牌模板样式
```

### 核心类
- `DeckTemplateSystem`：管理套牌模板功能的主类
  - 版本检测和选择
  - 数据加载和关联
  - 界面渲染和交互
  - 搜索过滤
  - 数据导出

## 界面预览

### 首页
- 两个大按钮：版本对比 和 套牌模板
- 清晰的功能选择界面

### 套牌模板页面
- 顶部：版本选择区域
- 中部：套牌统计摘要
- 搜索和过滤工具栏

- 采用与版本对比功能相似的设计风格
- 浅色主题，简洁明了
- 卡片式布局，便于浏览
- 响应式设计，支持不同屏幕尺寸

- 数据文件必须符合正确的格式
